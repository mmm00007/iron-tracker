## Fixes Applied (15 TypeScript errors across 13 files)

### 1. `NewSet` type missing DB-computed fields
**File:** `frontend/src/hooks/useSets.ts`

`NewSet = Omit<WorkoutSet, ...>` didn't exclude the new trigger/application-computed fields. Expanded the Omit to exclude: `updated_at`, `duration_seconds`, `distance_meters`, `distance_unit`, `training_date`, `side`, `rest_seconds`, `workout_cluster_id`.

### 2. Optimistic set construction missing new fields
**File:** `frontend/src/hooks/useSets.ts:111`

The optimistic placeholder `WorkoutSet` was missing all new fields. Added null defaults for each.

### 3. `EquipmentVariant.rating` not optional
**File:** `frontend/src/types/database.ts:106`

Made `rating` optional (`rating?:`) since variants can be constructed without a rating. This fixed 5 errors across `VariantBottomSheet.tsx`, `VariantChipRow.test.tsx`, `useVariants.ts`, `MachineIdentifyPage.tsx`, and `VariantManagerPage.tsx`.

### 4. `Profile` DEFAULT_PROFILE missing new fields
**File:** `frontend/src/hooks/useProfile.ts`

Added missing fields: `timezone: 'UTC'`, `day_start_hour: 4`, `height_cm: null`, `cluster_gap_minutes: 90`, `sleep_window_start/end: null`, `workout_window_start/end: null`.

### 5. `AnalysisReport` type import broken
**Files:** `frontend/src/hooks/useAnalysis.ts`, `frontend/src/pages/analysis/AnalysisPage.tsx`

The previous audit consolidated the duplicate `AnalysisReport` type but broke the import in `AnalysisPage.tsx`. Added re-export: `export type { AnalysisReport } from '@/types/database'` and restored `AnalysisInsight` interface. Fixed `insights` rendering with proper type cast since DB type is `Record<string, unknown>` while the component treats it as `AnalysisInsight[]`.

### 6. Test mock data missing `updated_at` and new fields
**Files:** `analytics.test.ts`, `deloadDetection.test.ts`, `sessionGrouping.test.ts`

All three test files' `makeSet()` helpers were missing the new `WorkoutSet` fields. Added null defaults.

### 7. Unused variables
- `SessionCard.tsx:176` — removed unused `exIdx` from `.map()`
- `LibraryPage.tsx:43` — removed unused `ExerciseWithMuscles` import

### 8. CSP blocking Google Fonts
**File:** `netlify.toml`

`font-src 'self'` blocked fonts from `fonts.gstatic.com`. `style-src 'self' 'unsafe-inline'` blocked CSS from `fonts.googleapis.com`. Added both origins to their respective CSP directives.
